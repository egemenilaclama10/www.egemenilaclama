# 🛡️ Egemen İlaçlama Web Projesi

Egemen İlaçlama için geliştirilen, kullanıcı dostu ve hızlı aksiyon odaklı kurumsal web sitesi çalışması.

## 🚀 Öne Çıkan Özellikler
* **Hızlı Teşhis:** Kullanıcıların haşere fotoğraflarını WhatsApp üzerinden uzmanlara gönderebilmesi.
* **Konum Paylaşımı:** Adres tarifi ile uğraşmadan tek tıkla konum paylaşımı.
* **Geniş Hizmet Ağı:** Kuzeyden güneye (Gökçeada'dan Dikili'ye) tüm bölge kapsamı.
* **Modern Arayüz:** Sabit navigasyon ve yüksek çözünürlüklü profesyonel görseller.

## 🛠️ Kullanılan Teknolojiler
* HTML5 / CSS3
* FontAwesome İkon Seti
* Google Fonts (Montserrat & Poppins)
